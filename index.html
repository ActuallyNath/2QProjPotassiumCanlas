PROCESS

The drawing tablet I use is a HUION H430P and I use the program
"Firealpaca", it's free and very beginner-friendly. I usually start off with a
3000 x 3000px canvas with 350 dpi, I sketch with the default watercolor
brush with either blue or red. Then I set the sketch on 50 percent opacity
and I add a layer above it and do my lineart with usually the same brush
except thinner and with darker color. I put my sketch layer on invisible and
create a layer folder underneath the lineart, I add in my flat colors and start
shading by adding clipping layers on top of the flat colored layers. With
shading I first locate my light source and shade according to it, I mainly use
the default brushes and I blend using the mix and blur brush. Finally, I
rasterize the file and add a clipping layer above everything, I fill it in with
purple and then set that layer to 'multiply' and the opacity to 17 percent,
after that, I add another clipping layer on 'add' and I add the lighting. Finally,
I add my signature, the date below the signature and I'm done, I sometimes
add minor adjustments but that's pretty much it, that's how I draw.
